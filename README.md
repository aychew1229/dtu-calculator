## 🚀 Key Features

### 📊 Professional Transcript Engine
* **Instant PNG Export:** Automatically generates and downloads a high-quality digital transcript of your results.
* **Universal Compatibility:** While optimized for DTU, the logic supports user-defined university names and department inputs.
* **Smart GPA Logic:** Handles complex credit hour weightings with 100% mathematical accuracy.

### 🛡️ Core Security & Integrity
* **Anti-Inspection:** Built-in protection scripts to prevent source code tampering and ensure result validity.
* **Privacy First:** All calculations are performed client-side; no student data is stored on external servers.

### 📱 Responsive Engineering
* **Mobile-First Interface:** Designed for quick use on smartphones during registration or grade-checking periods.
* **Lightweight Performance:** Zero-bloat code ensures fast loading even on limited network connections.

---

## 🛠️ Technical Implementation

* **Frontend Architecture:** Semantic HTML5 and modern CSS3 for a clean, academic aesthetic.
* **Logic Engine:** JavaScript (ES6) for real-time GPA calculation and dynamic DOM manipulation.
* **Imaging:** Integrated canvas-to-image logic for high-resolution PNG generation.

---

## 📋 User Guide

1.  **Identity Setup:** Enter your University Name, Student ID, and Department.
2.  **Course Entry:** Input your course titles and their respective credit hour values.
3.  **Process:** Click **Calculate** to see your Semester/Cumulative GPA.
4.  **Export:** Select **Download as PNG** to save your digital transcript for your records.
